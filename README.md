# 🏏 IPL Analytics Dashboard (Excel Project)

An interactive **IPL Analytics Dashboard** built entirely in Microsoft Excel to analyze player and team performances using dashboards, pivot tables, slicers, and data visualization techniques.

---

## 📌 Project Overview

This project provides deep insights into the Indian Premier League (IPL) data through two interactive dashboards:

- 👤 **Player Performance Dashboard**
- 🏆 **Team Performance Dashboard**

The dashboard helps users analyze player statistics, team performance, matches, and venues with a user-friendly interface.

---

## 🚀 Features

### 👤 Player Dashboard
✔️ Total Runs, Matches, Fours & Sixes  
✔️ Strike Rate Analysis  
✔️ Batting Performance Comparison  
✔️ Player-wise Filtering using Slicers  
✔️ Interactive Charts and KPIs  

### 🏆 Team Dashboard
✔️ Total Matches Played  
✔️ Wins & Losses Analysis  
✔️ Team Performance Comparison  
✔️ Venue-wise Performance Analysis  
✔️ Dynamic Filtering and Visualization
